Actors:

* User

SoftGoals:

* View a map display of public recycling bins around the user's area
	+ Specifications:
		- The system should display the location of public recycling bins on a map
		- The system should allow the user to filter the map by type of recyclable material accepted by each bin
		- The system should allow the user to search for recycling bins near their location
		- The system should provide information on the nearest recycling bins to the user's location
		- The system should display the number of recycling bins in each location
		- The system should allow the user to view the details of each recycling bin, including the type of material accepted and the location of the bin
* View safe disposal events currently being organised around the user's area
	+ Specifications:
		- The system should display a list of upcoming safe disposal events in the user's area
		- The system should provide information on the location, date, and time of each event
		- The system should allow the user to filter the list of events by type of disposal (e.g., electronics, hazardous waste, etc.)
		- The system should allow the user to search for events near their location
		- The system should provide information on the nearest event to the user's location
		- The system should allow the user to view the details of each event, including the location, date, and time of the event
* View all locations of recycling centers on a map, so that the user can check which routes to take to drop off waste
	+ Specifications:
		- The system should display the location of all recycling centers on a map
		- The system should allow the user to filter the map by type of recyclable material accepted by each center
		- The system should allow the user to search for recycling centers near their location
		- The system should provide information on the nearest recycling center to the user's location
		- The system should allow the user to view the details of each recycling center, including the type of material accepted and the location of the center

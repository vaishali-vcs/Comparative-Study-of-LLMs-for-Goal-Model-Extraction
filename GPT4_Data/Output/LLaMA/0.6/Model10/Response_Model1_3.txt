SoftGoals:

* Anonymity: The system should provide an anonymous platform for kids and youth to seek counseling.
	+ Specifications:
		- The system should not require users to provide personal information or identify themselves in any way.
		- The system should use end-to-end encryption for all communication.
		- The system should have a strict privacy policy that prohibits the sharing of user data with third parties.
		- The system should have a robust security framework to protect user data and prevent unauthorized access.
* Convenience: The system should provide convenient counseling services for kids and youth.
	+ Specifications:
		- The system should be accessible through multiple channels (text messaging, Cyber Café, portal, chat rooms).
		- The system should provide real-time counseling services.
		- The system should have a user-friendly interface that is easy to navigate.
* Promptness: The system should provide prompt counseling services for kids and youth.
	+ Specifications:
		- The system should have a responsive design that allows users to quickly access counseling services.
		- The system should have a reliable and fast communication infrastructure.
		- The system should have a dedicated team of counselors who are available 24/7.
